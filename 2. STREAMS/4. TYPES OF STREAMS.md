# TYPES OF STREAMS

## READABLE STREAM

PRODUCES DATA

YOU CAN PIPE FROM IT

```js
__readable__.pipe(A)
```

## WRITABLE STREAM

CONSUMES DATA

YOU CAN PIPE TO IT

```js
A.pipe(__writable__)
```

## TRANSFORM STREAM

CONSUMES AND PRODUCES TRANSFORMED DATA

```js
A.pipe(__transform__).pipe(B)
```

## DUPLEX STREAM

CONSUMES DATA SEPARATELY FROM PRODUCING DATA
